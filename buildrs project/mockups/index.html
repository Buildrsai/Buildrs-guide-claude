Notes de conception
— Chaque chapitre dans le dashboard est navigable via la sidebar. Le contenu change dynamiquement (JS).
— Gate Auth sur le passage Landing → Dashboard. Sans Google OAuth, aucun accès au contenu.
— CTA Final : 3 offres — Guide VibeCoder 17€ · Installation Claude 1:1 297€ · Guide complet 97€.
— Stack cible : React + TypeScript + Vite + Supabase Auth (Google) + Vercel. Route : /guide.
— Mobile-first. Les générateurs et guides sont consommés majoritairement sur mobile.
